## INSTALLATION

``` bash
$ git clone https://github.com/hiiruki/CheckRek
$ cd CheckRek
$ pip3 install requests
```

## USAGE

``` bash
$ python3 checkrek.py
```
Format:

- Bank      : 338801028216xxx
- E-Wallet  : 08123456789
